# Guess the number.

Esse codigo apenas foi feito com o intuito de melhorar minha 
experiencia com **C** e diversão.

Como jogar:
1- Nome do **player1** via teclado.
2- Nome do **player2** via teclado.
3- Um numero é sorteado de 1 a 10 pelo *sistema*.
4- Os **jogadores** tentam adivinhar.
5- Toda vez q os dois jogadores erram, o *sistema* da mais chances aos player, sem modificar o numero sorteado. 
6- Se alguem acertar, o *sistema* sorteia o codigo novamente, continuando o jogo.
